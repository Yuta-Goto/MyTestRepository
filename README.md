## おおまかな手順
１．あらかじめ必要なものをインストール：
・Node.js バージョン 8.0 以降
・Java JDK バージョン 11 以降
・Firebase CLI バージョン 8.14.0 以降

２．作業フォルダをfirebaseプロジェクトとして初期化
```rb
firebase init
```
